# APUSH Quiz Master

A quiz application for APUSH Period 4 study questions.

## How to Use Locally

Simply open `index.html` in your web browser.

## Files Included

- `index.html` - Main HTML file
- `script.js` - Quiz logic and functionality
- `style.css` - Styling
- `question1.js` through `question4.js` - Question sets (100 questions total)

## Sharing Options

### Option 1: GitHub Pages (Recommended)
1. Create a GitHub account at github.com
2. Create a new repository
3. Upload all files to the repository
4. Go to Settings > Pages
5. Select "main" branch and save
6. Share the link: `https://yourusername.github.io/repository-name/`

### Option 2: Share Files Directly
- Zip this entire folder
- Send via email, Google Drive, Dropbox, etc.
- Friend downloads and opens `index.html`

### Option 3: Local Network Sharing
- If on same WiFi network, use a local server
- Share your local IP address
